## 정은혜의 포트폴리오
Next.js와 TypeScript 기반으로 제작한 개인 포트폴리오 프로젝트입니다.

#### 주요 섹션 소개
- About Me: 경력, 수상 내역, 보유 기술 스택을 한눈에 확인할 수 있습니다.
- Reference: 추천서를 포함하여 신뢰도를 높였습니다.
- Projects: 개인 및 팀 프로젝트를 필터링하여 확인할 수 있습니다.

#### 기술 스택
- React
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
